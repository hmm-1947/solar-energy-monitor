# Solar Energy Monitor

A **solar and energy monitoring system** built around a **Growatt MIN 5000 XTL inverter**, using an **ESP32** to read inverter data via **Modbus RTU over RS485**, and a **Flutter dashboard** to visualize real-time and historical energy data stored in **Firebase Realtime Database**.

---
